# scoop-onair

Scoop bucket for [onair](https://github.com/dannotes/onair) — a small Rust binary that watches Microsoft Teams and turns a Philips WiZ smart bulb red while you're on a call.

## Install

      scoop bucket add onair https://github.com/dannotes/scoop-onair
      scoop install onair

## Updating

      scoop update onair

## Source

The actual code lives at https://github.com/dannotes/onair. Issues and PRs go there.

This repo only contains the auto-generated manifest. It's updated by a GitHub Actions workflow in the main repo on every `v*.*.*` tag.
